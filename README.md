# **AI-Assisted EHR Patient Data Validation Framework**

A healthcare quality engineering framework that validates FHIR R4 resources using Playwright, JSON Schema validation, and Ollama-powered AI analysis.
https://hl7.org/fhir/

#**Tech Stack**
- Playwright
- TypeScript
- FHIR R4 APIs
- Ollama
- JSON Schema
- GitHub Actions

#**Architecture**
- FHIR API
- Playwright
- Schema Validation
- Ollama Analysis

    
#**Features**
- FHIR resource validation
- JSON schema assertions
- AI-assisted healthcare data quality analysis
